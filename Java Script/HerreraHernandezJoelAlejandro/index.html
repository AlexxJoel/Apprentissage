Arreglos


  • Un arreglo es un conjunto de elementos de cualquier tipo de dato que acepte JS.
  • Tambien se puede decir que es una variable que puede almacenar varios valores.
  • Los arreglos empiezan desde el numero 0.
  • No tienen que estar especificado el numero de espacios que tendran el arreglo
  • El metodo length nos permite conocer cuantos objetos,espacios tiene una arreglo

  • Se asigna de la siguiente manera :var Nombre del arreglo =["Aqui van los elementos"];
  • Ejemplo var amigos =[ "Joel","Miguel"];

  • Para poder acceder a estos arreglos es: document.write(Nombre del arreglo[la posicion del elemento])
  • Ejemplo: document.write(amigos[0])
  • Nos mostraria : Joel .

  • Para poder cambiar el elemento que tiene el arreglo o solamente aumentar mas elementos es:
  • Nombre del arreglo[la posicion del elemento]="Elemento que vas a agregar o cambiar";
  • Ejemplo Remplazando elemento;
  • amigos[0]="Luis";
  • El arreglo ya seria : array ["Luis","Miguel"]
  • Ejemplo agregando:
  • amigos[2]="Joel";
  • Finalmente quedaria: array ["Luis","Miguel","Joel"]
