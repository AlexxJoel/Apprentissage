¿QUÉ ES EL OBJETO WINDOW?

El objeto window representa la ventana que contiene un documento DOM, cada pestaña contine su propio window objec

 PROPIEDADES  

  1. Closed  :Esta propiedad indica si la actual ventana esta cerrada o abierta.
  2. Defaultstatus  ;Obtiene / establece el texto de la barra de estado para la ventana dada.
  3. Frames  :Devuelve una matriz de subtramas en la ventana actual
  4. History  :Retorna una referencia a la historia del objeto.
  5. Length  :Retorna el numero de marcos en la ventana
  6. Location  :Obtiene/fija la location, o URL actual, del objeto de la ventana.
  7. Name  ;Obtiene / establece el nombre de la ventana.
  8. Opener:Devuelve una referencia a la ventana que abrió esta ventana actual.
  9. Parent  ;Devuelve una referencia al padre de la ventana o subtrama actual.
  10. Self:Devuelve una referencia de objeto al objeto de la ventana en sí.
  11.  Status :Obtiene / establece el texto en la barra de estado en la parte inferior del navegador.

 ¿QUÉ ES EL OBJETO DOCUMENT?

La interfaz Document representa cualquer página web cargada en el navegador y sirve como punto de entrada al contenido de la página (El árbol DOM). El DOM incluye elementos como <body> y <table>), entre muchos otros, y proporciona funcionalidad que es global al documento, como obtener la URL de la página y crear nuevos elementos en el documento

PROPIEDADES

  1.  Alinkcolor  :Devuelve o establece el color de los enlaces activos en el cuerpo del documento.
  2. Anchors  : Devuelve una lista de todos los anclajes en el documento.
  3. Applets  :Devuelve una lista ordenada de los applets dentro de un documento.
  4. Bgcolor  :Obtiene / establece el color de fondo del documento actual.
  5. Cookie  :Devuelve una lista de cookies separadas por punto y coma para ese documento o establece una sola cookie.
  6. Domain  :Devuelve el dominio del documento actual.
  7. Embeds  :Devuelve una lista de los <embed>elementos incrustados en el documento actual.
  8. Fgcolor :Obtiene / establece el color de primer plano, o color de texto, del documento actual. 
  9. Forms  :Devuelve una lista de los <form>elementos dentro del documento actual.
  10. Images  : Devuelve una lista de las imágenes en el documento actual
  11. Lastmodified  :Devuelve la fecha en la que el documento se modificó por última vez.
  12. Linkcolor  :Obtiene / establece el color de los hipervínculos en el documento.
  13. Links  : Devuelve una lista de todos los hipervínculos en el documento.
  14. Location  :Devuelve el URI del documento actual.
  15. Referrer  :Devuelve el URI de la página que se vinculó a esta página
  16. Title  : Devuelve el título del documento actual.
  17. Vlinkcolor  : Obtiene / establece el color de los hipervínculos visitados
  18. URL :Devuelve una cadena que contiene la URL del documento actual
